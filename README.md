# Customer Churn Prediction

## Project Overview
This project focuses on predicting customer churn using machine learning.
Customer churn refers to customers who stop using a company’s service.

## Problem Statement
Businesses lose revenue when customers leave. The goal of this project
is to identify customers who are likely to churn.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn
