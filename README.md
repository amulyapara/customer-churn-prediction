# Customer Churn Prediction

## Project Goal
Predict which customers are likely to churn using the Telco Customer Churn dataset.

## Dataset
Original dataset: WA_Fn-UseC_-Telco-Customer-Churn.csv

## Steps
1. Data cleaning & preprocessing
2. Feature preparation
3. Scaling & encoding
4. Logistic Regression model training
5. Model evaluation (accuracy, confusion matrix, classification report)
6. Feature importance visualization

## Results
- Accuracy: ~82%
- Confusion matrix and classification report included in the notebook

## Libraries
- pandas
- scikit-learn
- matplotlib
